Bermuda Bot
A Discord bot made for Bot Nation.


Command list

Command Description Usage
b.ban Bans the mentioned user. b.ban [mention] [reason]
b.help Displays all the available commands for your permission level. b.help [command]
b.kick Kicks the mentioned user. b.kick [mention] [reason]
b.lockdown This will lock a channel down for the set duration, be it in hours, minutes or seconds. b.lockdown [duration]
b.mute Mutes or unmutes a user for whatever reason. b.mute/b.unmute [mention] [reason]
b.ping Ping/Pong command. I wonder what this does? b.ping
b.purge Purges X amount of messages. Can only be used by mods b.purge [number]
b.reason Updates an unset moderator action. b.reason [case number] [new reason]
b.reload Reloads the command file, if it's been updated or modified. b.reload [command]
b.setgame Sets the bot's current playing status(Bot Owner) b.setgame [name]
b.unban Unbans the mentioned user b.unban [user] [reason]
b.urban Searches urban dictionary for the specified word b.urban [word]
b.warn Issues a warning to the mentioned user. b.warn [mention] [reason]

Note: Some of these commands are only available on the beta version of the bot.
